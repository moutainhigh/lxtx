一、游戏玩法说明

1. 每局游戏的投注时间为30秒，玩家在投注时间内首先选择自己要投的金币，然后点击要投注的区域进行投注，在投注时间外，不可进行投注。

2. 在一局游戏中，玩家可选择不同的金币进行投注，当玩家金币不足时，则无法投注，请充值金币。

3. 在屏幕的右下角有续压投注按钮，点击续压上一局所下的所有投注记录，当金币不足时，则续压无效。

4. 投注时间结束后，进入开奖时间，在此时间内，玩家等待开奖，不可投注也不可撤消投注。

5. 开奖结果，当转盘全部停下来后的箭头指向的区域即为本局游戏的开奖结果，与开奖结果匹配的高亮框显示 ，投注区域上则有亮框显示。

6. 开奖时间结束后，进入结息时间，系统会根据开奖结果，返还中奖者的投注金币和相应的奖励金币。至此，一局游戏结束，进入下一局游戏。

7. 左上区域显示上期开奖结果，点击后显示近期的开奖结果。

8. 如需查看历史投注记录，可在右上角菜单-投注记录中查询。

9. 如果玩家身上携帶的金币超过 20000000(2000万),可申请成为游戏庄家，成为庄家后，可最多连庄20局。

二、投注区域及赔率说明

1. 游戏玩法：当开奖结果与你所投注区域相同则为中奖。

2. 游戏赔率：

① 天地：每局会开出“天”或“地”中的任意一个，赔率为1赔1.96。

② 五行：每局会开出“金、木、水、火、土”中的任意一个，赔率为1赔5。

③ 十二生肖：每局会开出“鼠、牛、虎、兔、龙、蛇、马、羊、猴、鸡、狗、猪”中的任意一个，赔率为1赔12。

④ 二串一：“天地”玩法与“十二生肖”玩法结合，每一小时会显示一个二串一的组合，例如“天龙”赔率为1赔24，在开奖结果同时出天“天”和“龙”时，则中奖。

⑤ 三串一：“天地”玩法、“十二生肖”与“五行”玩法结合，每天显示一个三串一组合，例如：“天火龙”，赔率1赔120，当开奖结果同时出现“天”“火”“龙”时，则中奖。
