一、游戏玩法说明

1. 每局游戏的投注时间为30秒，玩家在投注时间内首先选择自己要投的金币，然后点击要投注的区域进行投注，在投注时间外，不可进行投注。

2. 在一局游戏中，玩家可选择不同的金币进行投注，当玩家金币不足时，则无法投注，请充值金币。

3. 在屏幕的右下角有续压投注按钮，点击续压上一局所下的所有投注记录，当金币不足时，则续压无效。

4. 投注时间结束后，进入开奖时间，在此时间内，玩家等待开奖，不可投注也不可撤消投注。

5. 开奖结果，当转盘停下来后光圈闪动区域为本局游戏的开奖结果，与开奖结果匹配的高亮框显示 ，投注区域上则有亮框显示。

6. 开奖时间结束后，进入结算时间，系统会根据开奖结果，返还中奖者的投注金币和相应的奖励金币。至此，一局游戏结束，进入下一局游戏。

7. 右边区域显示上期开奖趋势图。

8. 如果玩家身上携帶的金币超过 10000000(1000万),可申请成为游戏庄家，成为庄家后，可最多连庄20局。

二、投注区域及赔率说明

1. 游戏玩法：当开奖结果与你所投注区域相同则为中奖

2. 游戏赔率：

①　开出“宝马、奥迪、奔驰、大众”中的任意一个，赔率为1赔5

②　开出“兰博基尼、法拉利、劳斯莱斯、宾利”中的任意一个，赔率为1赔20

投注项目 赔率
宝马、奥迪、奔驰、大众
 1赔5
兰博基尼、法拉利、劳斯莱斯、宾利
 1赔20

三、上下庄说明
上庄条件：各闲家有申请上庄的机会，如果所持游戏币符合做庄条件则立即申请成功，等待坐庄的闲家信息将出现在“等待上庄”列表中。多名闲家申请，先申请者先成功。

下庄条件：

①　庄家游戏币不足上庄游戏币则强制下庄。

②　当庄家连续坐庄20次则强制下庄。

四、游戏计分说明

玩家下注所获欢乐豆以所押区域赔率进行结算。

断线计分

若有玩家或庄家断线，则游戏继续按照正常游戏行，也按照正常结束进行结算。

逃跑计分

庄闲逃跑，不退还玩家金币。都是正常结算输赢。
