高收入都来源于8小时外，你想赚更多吗？

2016-11-14 九州微云
    上天给予每个人的时间都是24小时，可并不是每个人都能获得同等的成就。


    每天平均工作8小时，睡觉8小时，那么剩下的可支配的8小时就显得尤为重要。如何利用“另外8小时”来赚更多的钱，让自己的生活更上一层楼？

      工作和睡觉只能让你活下去，但不能让你活得更好。所有对你有意义的事情，都是在剩下的8小时中完成的。

    对于工作不满意的人，通过在另外8小时中从事自己喜欢的事情并获得一定的成就，即可弥补工作所带来的挫败感，人生也更加积极和有意义。


可是，如何有效利用好另外八小时？


【1】少做一些不重要或无益的工作

要敢于对消耗你时间的人或者工作说不，如果实在推脱不了，必须要在日程表上加入这项活动，则必须从中删去另外一项不那么重要的活动以保持平衡，从而保证最重要的事情得到保障。

【2】和其他人结盟，运用他们的时间

当你没有钱的时候，可以跟邻居或者朋友合作一起共同完成不同的事情，各司其职。当有钱的时候，可以出钱请专业人士来完成从而腾出更多的时间。

【3】排除会耗损时间和精力的活动

列出你平时的每项活动及所耗费的时间；将活动按照递延、消除、减少或保持这4类进行分类;重新规划你的时间和精力。变得更有效率。善用科技和一些方法能更好地提升我们的时间利用率。

【4】赚得更多财富

时间已经有了，那我们到底怎样利用时间来赚得更多的财富呢？大家白天有一份全职工作，但他们想要拥有更多。光是节流，却不开源，紧巴巴的日子，人生多么无趣啊。
