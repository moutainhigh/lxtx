大学生一年内买了宝马，究竟是怎么做到的呢？

2016-12-24 微云九州盘

 作为一名在读的大二学生，我热爱大学生活，我的成绩不错，考试总是在班里名列前茅，但是有件事情一直困扰着我。因为我家条件不好，我上大学需要学费和生活费，这给家里增添了许多的负担，可是父母总是劝我安心读书，不要为钱的事情发愁，他们不让我去打工，怕影响我的学习和以后的前程。
我非常的爱我的父母，看着他们逐渐年老，而我还得向家里要学费和生活费，我感到非常的不忍和愧疚，常常为此苦恼，我曾经背着他们去打工，但是经过一段时间我辞职了，一是占用太多时间影响学业，二是收入不高，虽然也些帮助，但并不是很大。我有时一个人走在学校的操场上，默默的想念着家中的父母也在思考着到底如何能帮助家里分担更多的生活负担。

前一段时间，和几个志同道合的同学一起聊天，听见他们说现在有一种非常流行的金融衍生品叫云平台，投入小、获利快。他们劝我去了解一下，我一开始就回绝了，我不懂金融，又没有多少钱，没法做这个。后来，其中的一个和我非常熟的同学多次向我推荐，还把他的实际情况给我看，我不禁心动了。经过深入了解才知道，云平台原来和股票什么的不一样，这个只需要预测方向，一次最低投入10就可以了，而且随时随地都能购买。一开始，我把打工攒的4000元钱投入到市场里面去，一周时间，我已经挣了快70000元了。

盈利截图

               

随着时间的推移，我的操作越来越稳定，收益也越来越多，现在我每月的收益也在稳步增加，我终于自己挣出了学费和生活费，不用再向家里要钱了。同时，经过这一段时间的积累，还经常给父母买各种他们喜欢的东西，看着他们笑容，令我感到非常的满足。
